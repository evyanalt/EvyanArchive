Linux Commands

Here are some common Linux commands you need to know:

pwd
Shows your current directory.
ls
Lists files.
ls -la
Lists all files (including hidden ones) in a detailed format.
cd foldername
Move into a folder.
cd ..
Go up one folder.
cd ~
Go to your home directory.
cp file.txt backup.txt
Copy a file.
mv old.txt new.txt
Rename or move a file.
rm file.txt
Delete a file.
rm -r folder
Delete a folder and everything inside it.
mkdir newfolder
Create a folder.
touch file.txt
Create an empty file.
cat file.txt
Display a file.
less file.txt
View a file page by page.
nano file.txt
Simple terminal text editor.
grep "search" file.txt
Search for text.
htop
Interactive task manager.
free -h
RAM usage (human-readable).
df -h
Disk usage (human-readable).
uname -a
Kernel/system information.
neofetch
Pretty system info (if installed).
